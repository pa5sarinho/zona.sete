# Mapa

Num tamanho de tile de 45x45 (45cm²), forma-se comprimento e largura diagonais de 74x74 tiles, o que equivale a 33 metros, e comprimento e largura vertical e horizontal (64cm/tile) de 33x35 tiles equivalentes a 21x22 metros. Resumindo: num cenário em que os tiles têm 45cm² (ou 15voxel³), essa é a área visível aproximada na tela:

```txt                 
 _____________________ 
|                     |
|   diag: 33       22 |
|                   | |
|     21 -->        v |
|_____________________|
```
